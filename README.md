發想：  
在清大附近有兩間威秀影城，清大的學生們若想看電影，必須得兩座影城的資訊都查，查完一座的資訊卻又忘記另一座的，  
若我們能直接查看在所選日期，兩座影城有上映所選電影的時段，那也許會方便許多，所以生出了這個小專案。
